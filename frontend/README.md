# Interview Preparation Guide – React Website

A full-featured interview prep website powered by the Java scraper backend.

## Tech Stack
- **Frontend**: React 18 + Vite + Tailwind CSS + React Router v6
- **Code highlighting**: react-syntax-highlighter (Prism)
- **Data**: JSON files generated by the Java scraper

---

## Quick Start

### Step 1 – Install Node.js
Download and install Node.js LTS from https://nodejs.org (v18 or higher).

### Step 2 – Generate the data (run the Java scraper)
From the root of the project:
```bash
mvn package
java -jar target/webscraper-1.0.0.jar
```
This scrapes interview.in28minutes.com and writes:
- `frontend/public/data/index.json` + `frontend/public/data/{topic}.json`
- `frontend/public/pdfs/{topic}_Interview_Guide.pdf`

### Step 3 – Install frontend dependencies
```bash
cd frontend
npm install
```

### Step 4 – Start the dev server
```bash
npm run dev
```
Open http://localhost:5173 in your browser.

---

## Build for Production
```bash
cd frontend
npm run build       # outputs to frontend/dist/
npm run preview     # preview the production build locally
```

- 📱 Fully mobile responsive
- ⬅️ ➡️ Previous / Next section navigation
